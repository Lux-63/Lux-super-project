# Lux Super Project

Учебное портфолио Frontend-разработчика. Набор приложений, реализованных на чистом JS и перенесённых на Vue 3 для отработки навыков рефакторинга, работы с Canvas, API и оптимизации.

## Стек
- Vue 3 (Composition API)
- Vue Router, Pinia
- Vite
- HTML5 Canvas
- REST API
- Bootstrap

## Модули проекта
1. **Калькулятор** — поддержка дробей, отрицательных чисел, приоритета операций.
2. **Игры** («Виселица», «Найди клад», «Игра жизни») — логика, Canvas, оптимизация.
3. **Галерея собак** — работа с внешним API, два варианта UI.
4. **Прогресс-бары** — 3 варианта реализации, обработка состояний.
